# pure-text-capture
适用于Windows的轻量、无干扰的沉浸式文本采集器。
 (PowerShell)。

## 🎯 核心痛点与解决方案

在采集文本数据，如查阅大量复杂文档、整理素材或构建提示词库时，频繁在浏览器和记事本之间使用 `Alt+Tab` 切换不仅打断思路，还容易造成排版混乱。

**Pure Text Capture** 将采集动作缩减到极致：**你只需要专注阅读，按下 `Ctrl+C`。**

* **完全后台静默：** 脚本在后台监听剪贴板，自动捕获新内容并追加到本地文件。
* **彻底解决中文乱码：** 原生处理 UTF-8 编码，拒绝由于 PowerShell 默认编码导致的中文乱码问题。
* **视觉与心理反馈：** 提供带时间戳的高对比度控制台输出（Cyan/Yellow/Green），给予操作者明确的成功反馈。

## 🚀 极简使用指南

1. 下载本仓库中的 `Collect.ps1` 文件。
2. 右键点击该文件，选择 **使用 PowerShell 运行** (Run with PowerShell)。
3. 最小化控制台窗口，开始你的沉浸式阅读与复制。
4. 采集的数据将自动保存在桌面：`~/Desktop/Pure_Capture.txt`，段落间自动使用空行分隔。

## ⚙️ 内部逻辑说明

* 采用非阻塞的 `while($true)` 轮询机制。
* 动态比对剪贴板当前状态与上一状态，仅在内容发生实质性变化时进行 IO 写入，避免重复冗余数据。
